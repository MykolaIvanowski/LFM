what is $lt,$gt ?
$lt - lower than менше ніж (<)
$gt - greater than більше ніж  (>)
$eq: - value of a field equals the specified value (==)
$gte - field is greater than or equal to ( >= )
$lte - field is less than or equal to ( <= )
$in - вибирає документи, у яких значення поля дорівнює будь-якому значенню в указаному масиві.
	приклад { field: { $in: [<value1>, <value2>, ... <valueN> ] } }
$ne - is not equal (!=)
$nin -  select the field value is not in the specified array or the field does not exist.
